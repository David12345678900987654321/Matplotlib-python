## Aplicações da Biblioteca Matplotlib

Este projeto reúne diferentes aplicações da biblioteca Matplotlib para criação de gráficos e visualização de dados utilizando Python.

Foram desenvolvidos diversos tipos de gráficos, permitindo explorar diferentes formas de representação visual das informações e facilitar a análise dos dados.

### Gráficos desenvolvidos
- Gráfico de linha
- Gráfico de barras
- Gráfico de barras empilhadas
- Gráfico de dispersão
- Gráfico de pizza

O projeto tem como objetivo praticar técnicas de visualização de dados, personalização de gráficos e interpretação de informações por meio de representações visuais.

### Tecnologias utilizadas
- Python
- Matplotlib
